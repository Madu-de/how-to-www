# Kapitel 2.2.1 - CSS ID
Eine id ist ein HTML-Attribut, welches jedem Element gegeben werden kann. Beispiel:
```html
<p id="variableID"></p>
```
Diese ID sollte einzigartig sein. Bedeutet jede ID sollte nur einmal auf der ganzen Seite existieren. Beispiel:
```html
<p id="einzigartig"></p>
<p id="datumsanzeige"></p>
<img src="./assets/profile.png" id="profilbild"></img>
```
In CSS wird eine ID mit einem **#** vor dem ID-Namen selektiert. Beispiel:
```css
#datumsanzeige {

}
```
