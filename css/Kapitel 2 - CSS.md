# Kapitel 2 - CSS
CSS steht für Cascading Style Sheets und ist eine Beschreibungssprache, die für das Styling von HTML Elemente verantwortlich ist. Dabei kann jedes einzelne Element, aus einem HTML Code, angesprochen und gestylet werden.
