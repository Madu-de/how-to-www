# Kapitel 1 - HTML
HTML steht für Hypertext Markup Language und ist eine Beschreibungssprache, die den Aufbau einer Webseite beschreibt. **Es ist keine Programmiersprache**, da keine Kontrollstrukturen, wie Schleifen oder Verzweigungen, die Programmiersprachentypisch sind, existieren.
